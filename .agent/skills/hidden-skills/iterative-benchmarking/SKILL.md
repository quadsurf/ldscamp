# Benchmarking Workflow
1. **Baseline**: If Benchmark does not exist, create one using `examples/benchmark_template.py`.
2. **Execute**: Run Benchmark then apply changes.
3. **Validate**: Run Benchmarks again.
4. **Report**: Compare `baseline` vs `current` results in markdown.
